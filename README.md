# plotter.js
A library for plotting math functions on `<canvas>` (HTML element).

**[Live Demo](http://ja0n.github.io/plotter.js/)**

# Instructions
Clone this repo
```sh
git clone https://github.com/ja0n/plotter.js.git
```
Install dev dependencies
```sh
npm install
```
Run dev server
```sh
npm run serve
```
And open http://localhost:8090/

Or build files
```sh
npm run build
```
